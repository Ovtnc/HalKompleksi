# Hal Kompleksi Web Application

Hal Kompleksi uygulamasının modern web versiyonu.

## Kurulum

```bash
cd web
npm install
```

## Geliştirme

```bash
npm run dev
```

Uygulama http://localhost:3000 adresinde çalışacaktır.

## Build

```bash
npm run build
```

Build dosyaları `dist` klasörüne oluşturulacaktır.

## Özellikler

- ✅ Modern ve responsive tasarım (Tailwind CSS)
- ✅ Mobil-first yaklaşım
- ✅ Ürün arama ve filtreleme
- ✅ Ürün detay sayfaları
- ✅ Kullanıcı girişi ve kayıt
- ✅ Favori ürünler
- ✅ Piyasa raporları
- ✅ Profil yönetimi
- ✅ Logo ve branding

## Teknolojiler

- React 18
- TypeScript
- Vite
- React Router v6
- Tailwind CSS 3
- Modern UI/UX

## Tasarım

- **Responsive**: Mobil, tablet ve masaüstü için optimize edilmiş
- **Modern**: Tailwind CSS ile modern ve temiz tasarım
- **Accessible**: Erişilebilirlik standartlarına uygun
- **Fast**: Optimize edilmiş performans
